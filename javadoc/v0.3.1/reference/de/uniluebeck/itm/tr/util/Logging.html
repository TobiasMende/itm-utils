public static void setLoggingDefaults (Level level, Appender... appenders)

Sets the logging defaults according to the following algorithm:

  1. If the system property "log4j.configuration" is found and it points to a valid readable file the file is passed to log4j as a configuration file.
  2. If the first does not apply and a file with the name "log4j.properties" is found in the JVMs classpath this file is passed to log4j as a configuration file.
  3. If neither the first or the second apply log4j is configured to log level level and layout layout.

Parameters
level the log level to set on the root logger
appenders Appender instances to add to the root logger
